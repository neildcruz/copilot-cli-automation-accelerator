# Default User Prompt

Analyze the current codebase and provide a comprehensive assessment:

## 1. Overview
- Project structure and architecture summary
- Key technologies and frameworks identified
- Entry points and main components

## 2. Quality Assessment
- Code quality issues and potential bugs
- Performance concerns or optimization opportunities
- Security vulnerabilities or risky patterns
- Technical debt indicators

## 3. Recommendations
- Prioritized list of improvements (Critical/High/Medium/Low)
- Specific actionable fixes with code examples where applicable
- Quick wins that can be addressed immediately

Focus on the most impactful findings rather than exhaustive coverage. Limit the analysis to the top 10 most important items.
