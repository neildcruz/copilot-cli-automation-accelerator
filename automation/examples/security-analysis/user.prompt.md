# Security Analysis User Prompt

Conduct a thorough security audit of this codebase:

1. Scan all source files for security vulnerabilities
2. Check configuration files for security misconfigurations
3. Identify hard-coded secrets or credentials
4. Review authentication and authorization logic
5. Analyze input handling and validation
6. Check for insecure cryptographic practices
7. Identify dependency vulnerabilities

Generate a security report with severity ratings and remediation recommendations.
