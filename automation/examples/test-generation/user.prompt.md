# Test Generation User Prompt

Analyze the codebase and generate comprehensive tests:

1. Identify untested or under-tested modules and functions
2. Generate unit tests for all public methods and functions
3. Create integration tests for critical user workflows
4. Add tests for edge cases and error handling
5. Ensure tests follow project conventions and best practices
6. Calculate and report estimated code coverage
7. Generate test data fixtures where needed

Create test files in the appropriate test directory following project structure.
